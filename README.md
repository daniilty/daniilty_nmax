## DaniiltyNmax

Here's my implementation of a nmax funbox task

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'daniilty_nmax'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install daniilty_nmax

## Usage

    $ cat nums | daniilty_nmax 5
